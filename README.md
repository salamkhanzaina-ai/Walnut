# Walnut
Name: Walnut
Purpose: Teaches children ages 3-10 fundamental topics because the school curriculum bombards the students with subjects like math, science, english, second language, art, music, but we propose that the core subjects that kids at these ages should learn are logic (pure, foundational logic, critical thinking, reasoning, induction/deduction, pattern recognition, classification,etc), linguistics(the study of syntax, patterns in language, semantics, etymology, etc. Learning linguistics is very fundamental to learning languages as a whole), and finally, feelings(this topic will be changed into a better name, but the reason this subject is important is for multiple reasons; firstly, little children will be taught to manage their emotions, as they get older, the concepts become more complex like how to deescalate situations, different types of people, the second thing that feelings connect to is art and music, because the point of the arts is to invoke feelings in others)

Significance: Learning this foundational, core concepts, while learning from the regular school curriculum, creates a very well rounded, enriched, and intelligent child.

Research: At the ages of 2-10, children’s brains are very malleable, have very good learning capabilities, can learn difficult topics way quicker and more comprehensively than older people,)
Learning these topics, lay the foundation that will make learning math, science, languages, etc wayyy easier, and the child user will become more interested in learning too, not just application.  
(Aristotle quote about child until 7 years old)

Theme: Ancient greek, ancient fantasy/medieval fantasy, ancient egypt, etc
Why?: Ancient Greece was pioneers in logic, Ancient fantasy/Egypt we will create a fake undecipherable language that is reminiscent of Egyptian hieroglyphs, and for feelings, not sure yet, but we are thinking the settings will be in the clouds, where the weather represents emotions. 

Our approach: Gamified learning platform, very similar to Prodigy, but the topics will be similar to Brilliant.org. The games are going to look like this: The home page shows the child’s avatar, and then three pathways/doors/icons to choose for the three topics; feelings, logic, linguistics. If logic is pressed, then the user is taken on a road that looks like ancient ruins, where they can practice classification, and other practice simulations/fights, etc to learn logic. For linguistics, we can teach the user patterns and other stuff in linguistics and then take  them to this ancient tomb area with scrolls that are written in an ancient undecipherable language. We will give them some hints, and they are going to use their knowledge of linguistics to decipher the language to win a prize. There will be an element of sound added to learn sounds, consanants/vowels, etc, but not sure yet, For feelings, the user will be floating in the clouds. They can press how they feel which will change the weather to match the emotion(raining= sadness), and then our app will teach the user how to manage the emotion( umbrella, etc),  the feelings apart will also show a simulation of an ancient greek rally or some disagreement where there is an argument happening, and the user will learn how to de escalate the situation. They will also learn about art and how it invokes feelings, etc.
The topics in every category get more important according to the user’s age and progress(AI will monitor and analyse the user’s responses to allow the user to progress).


There will also be a curiosity bot fairy that the user can ask any question it has or curiosity. This will encourage curiosity


The app will also give them side quests:  Some side quests include for linguistics: Create a secret language and get 5 people to speak it fluently. For logic, i dont know yet but maybe like riddles or create a board game or something strategizing, etc,etc. For feelings; paint the feeling of seeing the colour red, or something like that. 

Another feature we want to implement is that as the user progresses, they must use their learnings to begin playing the game, kind of like prodigy, we want to add a strategy aspect and need to win. 
Also, as the user progresses, their avatar can also visibly grow.
